size

    사이즈 단위의 사용
  • 기본 단위는 'rem'과 '%' 단위 사용
  • html 태그에 font-size=75.0%로 설정 -> 1rem=12px로 상대값 지정함
  • 따라서 'rem'은 'px' 사이즈의 1/10로 간주하여 적용 (ex 12px=1.2rem)
  • 전체적인 사이즈 조정시, html 태그 font-size만 조정 (ex 480px 이하 font-size=75.0% (=12px))
    사이즈 단위의 예외적인 사용
  • 'em' : input, select, button의 height / 폰트 사이즈가 유동적인 FontAwesome icon의 margin
  • 'px' : max-width, min-width 등 일부 / 'layer-blocking-browser' 전체
